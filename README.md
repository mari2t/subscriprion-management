## Overview

サブスクリプションの料金や感想を管理するアプリです。  
This application manages subscription fees and your own impressions.

## Purpose of creation

1. サブスクリプションの一か月の総額、これまでの課金額を見える化したい。
2. サブスクリプションの料金、契約URL等を簡単に管理したい
3. 継続するかどうかの参考とするため、サブスクリプションを使用した感想を投稿したい
4. T3アプリをつくってみたい
